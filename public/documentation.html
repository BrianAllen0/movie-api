Get a list of all movies in the database

/movies  Method: GET

JSON Response:

{
 {movie1},
 {movie2},
 ...
}

Get a list of all movies marked as favorites

/movies/favorites  Method: GET

JSON Response:

{
 {movie1},
 {movie2},
 ...
}

Get data about a specific movie

/movies/title  Method: GET

JSON Response:

{
 Title: string,
 Description: string,
 Director: string,
 Genre: string,
 Featured: boolean
}

Get data about a movie genre

/genres/name  Method: GET

JSON Response:

{
 Name: string,
 Description: string
}

Get data about a director

/directors/name  Method: GET

JSON Response:

{
 Name: string,
 Bio: string,
 Birth: date,
 Death: date
 ...
}

Add a movie to your favorites list

/movies/favorites/add/title  Method: POST

Request Body:

{
 Username: string,
 Title: string
}

Login to a user account.

/user/login  Method: POST

Request Body:

{
 Username: string,
 Password: string
}

Update info in a user account.

/user/update  Method: PUT

Request Body:

{
 Username: string,
 Password: string,
 Email: string
}

Register a new user account.

/user/register  Method: POST

Request Body:

{
 Username: string,
 Password: string,
 Email: string,
 Birthday: string
}

Remove a movie from your favorites list

/movies/favorites/remove  Method: DELETE

Request Body:

{
 Username: string,
 Title: string
}
